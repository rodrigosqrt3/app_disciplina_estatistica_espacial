Este projeto consiste em um aplicativo Shiny que explora a biodiversidade do Brasil, utilizando dados da API do GBIF (Global Biodiversity Information Facility). O aplicativo permite a visualização e análise das espécies distribuídas em diferentes biomas do país. Pode ser acessado pelo link: https://3w6g1b-rodrigo-villa.shinyapps.io/Semin-1/
